Markdown-Essentials

Letâ€™s note down some markdown essentials here for reference.

1. Make things bold: text

**text**

2. Make things italic: italic

_text_

3. Make things italic and bold: italic

**_text_**

4. Headers (you can make words italic as usual)

# One
## Two
### Three
#### Four
##### Five
###### Six

5. Links (you can make words italic or bold as usual) looks like this 

First version to do it:
	[name of link as shown](url)

	Second version to do it:
	[name of link as shown][reference link]
	-- then at the end of the document
	[reference link]: url

6. Pictures

First version to do it:
	![verbal description of pic][url]

	Second version to do it:
	![name of link as shown][reference link]
	-- then at the end of the document
	[reference link]: url

7. Quotes (use italics and bold inside as usual)

Use the greater sign on each new paragraph that is still part fo the quote!

looks like this.

> quotable text

8. Lists (use italics and bold inside as usual)

  • Item one
  • Item two
  • Giant Elephant
  1. Numbered Item one
  2. Number Item two
  3. Numbered Giant Elephant
  • Major Item
    • Minor Item
      • More Minor Item
* Item one
	* Item two
	* Giant Elephant

	1. Numbered Item one
	2. Numbered Item two
	3. Numbered Giant Elephant

	* Major Item
		* Minor Item
			* More Minor Item

9. Line breaks

Hard break:

Done by seperating lines forcefully by an empty line

Soft break:..

Done by adding two spaces at the point where you desire linebreak (  ) (represented by two point above)
